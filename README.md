Switch
```js
<Switch expression='red' >
  <Case value='red'>
    Red
  </Case>
  <Case value='green'>
    Green
  </Case>
  <Case value='gray'>
    Gray
  </Case>
  <Default>
    Default
  </Default>
</Switch>
```
